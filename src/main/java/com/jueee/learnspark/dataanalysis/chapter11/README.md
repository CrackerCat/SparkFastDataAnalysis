### 第 11 章	基于 MLlib 的机器学习 ###
MLlib 是 Spark 中提供机器学习函数的库。它是专为在集群上并行运行的情况而设计的。  

MLlib 中包含许多机器学习算法，可以在 Spark 支持的所有编程语言中使用。
